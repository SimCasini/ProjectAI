Per riprodurre i risultati si devono utilizzare tutti i file .py e i file txt che corrispondono ai dataset.
Nei vari main.py sono presenti i nomi degli attributi dei dataset e i loro valori che attraverso la funzione test_plot() sono usati per creare i dataset.
Per ognuno dei dataset è presente un main.py su cui fare il run per riprodurre il risultato.
La struttura del dataset e dell’albero sono riprese dall' aima python repository su github e sono rappresentate rispettivamente nelle classi Dataset, Tree e Leaf.
I dataset sono presi dall' UCI Machine Learning Repository.
